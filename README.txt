Installation:

- Install Skyop's animated jetways if your using FlightGear 2.0
Download : http://flightgear.org/forums/viewtopic.php?f=5&t=8728

- If you use FlightGear git, either pull the jetways from the repo
or download : http://www.mediafire.com/?2ssjc3cbnbagn2h

- Extract the Animated jetways to your fgdata folder

- Now, copy the contents of the 'Put me inside your fgdata folder'
to your fgdata folder.
